# SKYG Template Academy

Academia online con Next.js 15, Supabase, Stripe y Vercel.

## Setup rápido

1. Importa este repo en Vercel
2. Agrega las variables de entorno (ver `.env.example`)
3. Ejecuta el SQL de la Fase 1 del ebook en Supabase
4. Haz deploy ✓

## Variables de entorno

Copia `.env.example` como referencia y agrega los valores en **Vercel → Settings → Environment Variables**.

## Stack

- **Next.js 15** App Router
- **Supabase** Auth + PostgreSQL + RLS
- **Stripe** Pagos + Webhooks
- **Vercel** Hosting + CI/CD
